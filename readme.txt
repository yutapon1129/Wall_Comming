＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
------------------------------------------------------------------------
  iPhone
------------------------------------------------------------------------
最初にiPhoneにUnity Remote 5というアプリをインストールして下さい。

1. Unityを起動してFileタブをクリック。

2. Fileタブの中にあるBuild Settingsをクリック。

3. Build Settings内のPlatformの中にiOSというものがあるのでクリック。

4. iOSをクリックした後、右にOpen Download Pageとあればそこをクリックしてインストールしてください。
   Open Download Pageと書かれてなければ特に何もしなくても大丈夫です。
　 インストールが終わったらUnityを再起動してiPhoneをPCに接続してUnity Remote 5を起動してください。

5. Unityのゲーム画面に戻って次に、Fileの横にあるEditタブをクリック。

6. Editタブの中にあるProject Settingsをクリック。

7. Project Settingsの中にあるEditorを開きます。

8. Editorの中にUnity Remoteという項目があり、その中にDeviceといった場所があると思います。

9. Deviceの中が恐らく最初 None になっていると思うので、その中をAny iOS Deviceに変えてください

10. Any iOS Deviceになっている状態でPlayを押すとiPhoneで動かせると思います。

------------------------------------------------------------------------
  Android
------------------------------------------------------------------------
最初にAndroidにUnity Remote 5というアプリをインストールして下さい。

1. AndoroidをPCに接続してUnityを起動します。

2. 起動したらFileの横にあるEditタブをクリック。

3. Editタブの中にあるProject Settingsをクリック。

4. Project Settingsの中にあるEditorを開きます。

5. Editorの中にUnity Remoteという項目があり、その中にDeviceといった場所があると思います。

6. Deviceの中が恐らく最初 None になっていると思うので、その中をAny Android Deviceに変えてください。

7. Any Android Deviceになっている状態でPlayを押すとAndroidで動かせると思います。
